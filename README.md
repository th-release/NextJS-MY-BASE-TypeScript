# th-release Next.JS -TypeScript
타입스크립트로 NextJS 사용하기

> JavaScript 버전을 원하시는 분은(https://github.com/th-release/NextJS-MY-BASE-Node)로 이동해 주세요

## 준비물
```* node v12.x 또는 그 이상
* yarn v1.x
```

## 사용법
### 1. 템플릿 사용

저장소를 만들고 NextJS-MY-BASE-TypeScript를 다운로드 한 후에 저장소 넣습니다.

### 2.필수 구성요소 다운로드
```yarn```\
로 구성요소를 다운로드 합니다

### 4. 실행
```yarn build & yarn start -p <포트>```\
으로 실행 시킬 수 있습니다.

### 5. 완료

이제 재밌
